<p><b>Privacy Preserving Identity Verification</b> uses blockchain and zero-knowledge proofs to verify identity claims while minimizing exposure of sensitive user data.</p>

<p>The project enables selective disclosure of attributes such as age, gender, and state of residence by wiring Circom circuits into Solidity contracts whose Groth16 proofs are orchestrated and verified locally through a Hardhat environment.</p>

### Tech Stack

- Solidity
- Circom
- SnarkJS
- Hardhat

### Highlights

- Designed and implemented a privacy-preserving identity verification system enabling selective disclosure of user attributes using zero-knowledge proofs.
- Developed Circom circuits and integrated Groth16 proofs with Solidity smart contracts, verified locally via the Hardhat environment.
